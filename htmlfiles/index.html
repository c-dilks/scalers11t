Scaler Counts and Corrections - Correction Equations [PDF]

  • For VPD, we only have VPDx scaler bit
  • The physical process probability for VPDX is thus equal to the scale count probability, since it is not possible to apply accidentals corrections, given that we don't have VPDE or VPDW scale counts
  • Multiples corrections are given by the same correction equation as for BBCX and ZDCX, but with N_E=N_W=0

Relative Luminosities

  • For a few runs, raw VPDX counts was zero; I've set the relative luminosity values for these runs equal to 1, but with large error bars; this was done so that the plots below draw correctly
  • An extra consistency check was then added to mark runs as bad if the rellum error bars are set this large, thus runs with no VPDX scales are marked as having inconsistent rellum measurements in the final output tree
